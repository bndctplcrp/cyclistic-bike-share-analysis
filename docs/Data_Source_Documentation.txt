Dataset Name:
Cyclistic (Divvy) Bike-Share Trip Data – 2019

Data Provider:
Motivate International Inc. (Divvy Bikes / Lyft Urban Solutions)

Source:
Divvy Trip Data Public Repository

Time Period Covered:
January 2019 – December 2019

Files Used:
- Divvy_Trips_2019_Q1.csv
- Divvy_Trips_2019_Q2.csv
- Divvy_Trips_2019_Q3.csv
- Divvy_Trips_2019_Q4.csv

File Format:
CSV (Comma-Separated Values)

Data Description:
This dataset contains historical bike trip records from the Cyclistic bike-share program 
(operating as Divvy in Chicago). Each record represents a single bike trip and includes 
information such as trip start and end times, ride duration, bike type, station locations, 
and rider membership type (casual or annual member).

Intended Use:
The data is used to analyze behavioral differences between casual riders and annual members 
in order to support marketing strategies aimed at increasing annual memberships.

Data Licensing:
This dataset is publicly available and provided under the Divvy Bikes Data License Agreement. 
The data is permitted for analysis and educational use.

Privacy and Security:
All personal identifiable information (PII) has been removed by the data provider. The dataset 
does not include rider names, contact details, or payment information, ensuring user privacy.

Data Credibility and Reliability:
The data is sourced directly from the official bike-share operator and is considered reliable, 
original, and credible. It follows ROCCC principles (Reliable, Original, Comprehensive, 
Current, and Cited).

Known Limitations:
- No demographic data about riders is included.
- Rider location and identity cannot be determined.
- Trips with missing or invalid timestamps may require cleaning during the Process phase.
