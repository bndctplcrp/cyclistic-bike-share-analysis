# Data Source Documentation

## Dataset Name
Divvy / Cyclistic Bike-Share Trip Data

## Data Provider
Motivate International Inc. (Divvy Bikes)

## Description
This dataset contains historical bike-share trip records from Cyclistic, a bike-share program operating in Chicago. Each record represents an individual bike trip, including ride duration, start and end stations, and rider type.
